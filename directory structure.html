The directory tree

  • chibios : you can download this from internet (version 2.6.3) unpack the fatfs zip in chibios!
  • firmware : the C firmware running on the board
  • objects : all the objects you can use in a patch. This is XML containing the relevant C-code and definition of inlets, outlets, parameters...
  • patch : here the current document is compiled
  • patches : example documents!
  • platform_xxxx : scripts and binaries for every platform. They are executed with the Axoloti root folder as working directory!
  • src : the GUI source code (in JAVA)
  • dist : the Java executable
